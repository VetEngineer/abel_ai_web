# ABEL AI Website Master Plan (v1)

## 1) 프로젝트 요약
- 목적: AEO/GEO/SEO 전문성 신뢰 확보 + 무료 진단 전환 최적화.
- CMS: WordPress + UiCore Pro.
- 언어: 국문 중심.

## 2) 핵심 목표/KPI (초기)
- 무료 진단 요청 전환율 상승.
- AEO/GEO 기반 질문형 섹션 완성.
- 2주 내 런칭.

## 3) 포지셔닝/메시지
- 포지션: AEO/GEO/SEO 통합 최적화 에이전시.
- 키워드: 신뢰 / 정확 / 전환.
- 타깃: 범용(모든 산업).
- 타깃: 범용(모든 산업).

## 4) 사이트 구조(IA)
- Home (1-page 구조)
- Service (AEO/GEO/SEO 상세)
- Cases/Insights (케이스 또는 인사이트)
- Contact (무료 진단 요청)

## 5) 페이지별 구성(요약)
- Home
  - Hero(3D 영상)
  - Value Statement
  - Service 3종
  - Differentiators
  - Process
  - Deliverables
  - FAQ
  - Final CTA
- Service
  - 서비스 정의 + 방법 + 결과
- Cases/Insights
  - 사례 1개 + 인사이트 3개
- Contact
  - 진단 요청 폼/CTA

## 6) 디자인 시스템(요약)
- 컬러: Space Blue / Purple / Rich Black
- 히어로: 중앙 카피 + 배경 3D 루프 영상
- 텍스트 톤: 공격적/임팩트 중심

## 7) AEO/GEO/SEO 체크리스트(요약)
- 질문형 섹션 포함
- FAQ 간결 답변
- 스키마(Organization, WebSite, WebPage, Service)
- 전환 중심 CTA

## 8) 성능/접근성
- WebM + MP4 + Poster
- Lazy load
- 대비 확보

## 9) 런칭 체크리스트(요약)
- 모바일/데스크탑 QA
- CTA 동작 확인
- sitemap 제출
