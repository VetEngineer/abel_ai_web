# CTA Flow (v1)

## 기본 CTA
- 텍스트: 무료 진단 요청
- 위치: Hero + Final CTA + Service/Insights 하단

## 흐름
1) CTA 클릭
2) Contact 페이지 이동
3) 진단 요청 폼 제출
4) Thank-you 메시지 표시
5) 운영자 이메일 알림 + 리드 저장

## 폼 필드(최소)
- 이름
- 회사명
- 웹사이트/도메인
- 이메일
- 연락처
- 문의 내용(선택)

## 운영 원칙
- 응답 SLA: 24–48시간 내 회신
- 스팸 방지: reCAPTCHA 또는 간단한 허니팟
