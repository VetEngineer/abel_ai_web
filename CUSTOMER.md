# CUSTOMER.md — ABEL AI

## 회사 개요

- **회사명**: ABEL
- **포지션**: AEO/GEO/SEO 통합 최적화 에이전시
- **핵심 메시지**: "답변에 없으면 매출도 없습니다"
- **키워드**: 신뢰 / 정확 / 전환
- **설립**: 신규법인 (2025)
- **웹사이트**: https://abelai.kr
- **주요 서비스**: AEO(Answer Engine Optimization), GEO(Generative Engine Optimization), SEO(Search Engine Optimization)

|------|------|

## 고객 페인포인트

### 핵심 문제

1. **답변형 검색에서 노출 부족**: ChatGPT, Perplexity 등 생성형 AI에서 브랜드가 언급되지 않음
2. **콘텐츠는 많은데 전환이 약함**: 블로그, SNS 콘텐츠를 열심히 만들지만 실제 전환(문의/매출)으로 이어지지 않음
3. **기존 SEO만으로는 불충분**: 구글 검색 1등이어도 AI 답변에 포함되지 않으면 유입이 줄어드는 추세
4. **구조화된 콘텐츠 전략 부재**: 질문형 쿼리, 근거 중심 구조 등 AI 엔진이 선호하는 포맷을 모름

### 감정적 페인

- "경쟁사는 ChatGPT에 나오는데 우리는 안 나온다"
- "마케팅비는 쓰는데 성과가 보이지 않는다"
- "SEO 업체를 썼는데 뭐가 달라진 건지 모르겠다"
- "AI 검색이 대세라는데 뭘 어떻게 해야 하는지 모르겠다"

## 고객 현재 상태 (Before)

- 웹사이트는 있지만 검색/답변 최적화가 안 되어 있음
- 블로그나 SNS 콘텐츠를 운영하지만 구조화되지 않음
- ChatGPT/Perplexity에서 자사 브랜드 검색 시 언급되지 않음
- 전환(CTA) 동선이 불명확하거나 최적화되지 않음
- AEO/GEO 개념을 잘 모르거나, 알아도 실행 방법을 모름

---

## 고객 목표 상태 (After)

- ChatGPT, Perplexity, Google AI Overviews에서 브랜드가 인용됨
- 질문형 검색에서 직접 답변으로 노출
- 콘텐츠가 구조화되어 AI 엔진이 인용할 수 있는 "출처"로 기능
- 전환 동선이 최적화되어 유입→문의 전환율 상승
- 주간 업데이트 체계로 지속적 노출 유지

## 고객이 신뢰하는 신호

1. **구체적 프로세스**: 막연한 약속이 아닌, 진단→설계→실행의 명확한 3단계
2. **무료 진단**: 리스크 없이 현재 상태를 파악할 수 있는 진입점
3. **근거 중심 접근**: "근거 중심 콘텐츠 설계", 데이터 기반 의사결정
4. **주간 업데이트**: 지속적 관리와 개선 체계
5. **전문 용어 활용**: AEO/GEO/SEO 전문 지식 보유 입증
6. **빠른 응답**: 24-48시간 내 회신 SLA

## 구매 여정

```
인식 → 탐색 → 비교 → 문의 → 계약
```

1. **인식**: AI 검색에서 자사가 노출 안 된다는 것을 깨달음
2. **탐색**: AEO/GEO/SEO 관련 정보를 검색, ABEL AI 콘텐츠 발견
3. **비교**: 다른 SEO 에이전시와 비교, ABEL AI의 AEO/GEO 차별점 확인
4. **문의**: 무료 진단 요청 (핵심 전환 포인트)
5. **계약**: 진단 결과 기반으로 범위 확정, 프로젝트 시작

## 경쟁 환경

### 직접 경쟁
- 기존 SEO 에이전시 (AEO/GEO 미지원이 대부분)
- 콘텐츠 마케팅 에이전시

### 간접 경쟁
- 인하우스 마케팅 팀 (직접 학습하여 대응)
- 종합 디지털 마케팅 에이전시
- AI 콘텐츠 생성 도구 (Jasper, Copy.ai 등)

### ABEL AI 차별점
- **AEO + GEO + SEO 통합**: 개별이 아닌 통합 최적화
- **답변형 구조 전문**: AI 엔진이 인용할 수 있는 구조 설계 전문
- **근거 중심 접근**: 감이 아닌 데이터와 근거 기반
- **전환 중심**: 노출이 아닌 전환까지 설계
